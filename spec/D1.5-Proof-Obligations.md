## T1 — Decidability of Typing and Evaluation

**Discharges:** D0 Assumption 4 (static analyzability); D1.2 metatheorem.

$$
\forall\, \Gamma\, e.\ \ \mathrm{Decidable}\big(\exists \tau.\ \Gamma \vdash e : \tau\big)
\qquad\wedge\qquad
\forall\, \rho\, e.\ \ \mathrm{Decidable}\big(\exists v.\ \rho \vdash e \Downarrow v\big)
$$

```lean
theorem typing_decidable (Γ : Ctx) (e : Expr) : Decidable (∃ τ, HasType Γ e τ) := by sorry
theorem eval_terminates  (ρ : Env) (e : Expr) : ∃ v, BigStep ρ e v := by sorry
```

## T2 — Purity of the Algebra (Invariant I1)

**Discharges:** I1 (read-only algebra).

$$
\forall\, \rho\, e\, v.\ \ \rho \vdash e \Downarrow v \;\Longrightarrow\; \mathrm{writes}(e) = \varnothing
$$

```lean
theorem algebra_pure (ρ : Env) (e : Expr) (v : Val) (h : BigStep ρ e v) : writes e = ∅ := by sorry
```

## T4 — Registry Semantic Inertness (Invariant I4)

**Discharges:** I4. For any bijective token renaming $\sigma$, verdicts are stable.

$$
\forall\, \sigma\ \text{bijective on } \mathrm{dom}(R),\ \forall\, \bar\eta.\ \
\mathrm{verdict}(\hat{K}, \bar\eta) = \mathrm{verdict}(\sigma\!\cdot\!\hat{K}, \bar\eta)
$$

```lean
theorem registry_inert (σ : Token ≃ Token) (K : KB) (tr : Trace) :
    verdict K tr = verdict (rename σ K) tr := by sorry
```

## T5 — Presentation Erasure (Invariant I5)

**Discharges:** I5. Deleting $\hat{P}$ preserves all verdicts.

$$
\forall\, \hat{K}\, \bar\eta.\ \ \mathrm{verdict}(\mathrm{erase}_{\hat P}(\hat{K}), \bar\eta) = \mathrm{verdict}(\hat{K}, \bar\eta)
$$

```lean
theorem presentation_erasure (K : KB) (tr : Trace) :
    verdict (erasePresentation K) tr = verdict K tr := by sorry
```

## C1 — Minimality of $\langle B, T \rangle$ (Conjecture)

**Discharges:** D0 Hypothesis 2. No proper sub-basis $B' \subsetneq B$ spans all verifiable
corpora. Stated as an *irreducibility* conjecture; expected to be validated empirically
(D0 §8) rather than proved outright.

$$
\forall\, B' \subsetneq B.\ \ \exists\, \text{corpus } C.\ \ \neg\,\mathrm{representable}(C, \langle B', T \rangle)
$$
