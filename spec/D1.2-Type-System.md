# D1.2 — Type System

**Layer:** $\hat{K}$ / $\hat{S}$ · **Status:** DRAFT · **Depends:** D1.1

A many-sorted, monomorphic type system. Judgments are decidable (D1.5 §T1). Typing
contexts $\Gamma$ map variables to base types. There is no arrow type in $T$: values
are first-order (Invariant I3, kernel closure).

## 1. Base Types (Sorts)

$$
\tau \;::=\; \mathsf{Ent} \mid \mathsf{Iri} \mid \mathsf{Tok} \mid \mathsf{Act} \mid \mathsf{Unit} \mid \mathsf{Val} \mid \mathsf{Bool} \mid \mathsf{Time} \mid \mathsf{Sign} \mid \mathsf{Force} \mid \mathsf{State}
$$

Sorts are inhabited by the primitive domains of D1.1. $[\![\mathsf{Ent}]\!] = \mathbb{E}$,
$[\![\mathsf{State}]\!] = \Sigma$, etc. Sort assignment is total and syntax-directed.

## 2. Roles

A **role** annotates a positional argument of a constructor with its normative function.

$$
\rho \;::=\; \mathsf{bearer} \mid \mathsf{cparty} \mid \mathsf{holder} \mid \mathsf{operand} \mid \mathsf{target} \mid \mathsf{source} \mid \mathsf{cond} \mid \mathsf{body}
$$

**Role sorting.** $\mathrm{sort}(\mathsf{bearer}) = \mathrm{sort}(\mathsf{cparty}) = \mathrm{sort}(\mathsf{holder}) = \mathsf{Ent}$;
$\mathrm{sort}(\mathsf{cond}) = \mathsf{Bool}$; $\mathrm{sort}(\mathsf{target}) = \mathsf{Val}$.

## 3. Judgment Forms

$$
\Gamma \vdash e : \tau \qquad(\text{term } e \text{ has sort } \tau)
$$
$$
\Gamma \vdash_{\!B} k \;\;\mathsf{ok} \qquad(\text{kernel instance } k \text{ is well-formed})
$$

## 4. Structural Rules

$$
\frac{x : \tau \in \Gamma}{\Gamma \vdash x : \tau}\ \textsc{(T-Var)}
\qquad
\frac{v \in [\![ \tau ]\!]}{\Gamma \vdash v : \tau}\ \textsc{(T-Const)}
$$

$$
\frac{\Gamma \vdash e_1 : \mathsf{Val} \quad \Gamma \vdash e_2 : \mathsf{Val} \quad {\oplus} \in \mathrm{Cmp}}{\Gamma \vdash (e_1 \oplus e_2) : \mathsf{Bool}}\ \textsc{(T-Cmp)}
$$

## 5. Constructor Typing (Well-Formedness)

$$
\frac{
  \Gamma \vdash b : \mathsf{Ent} \quad
  \Gamma \vdash c : \mathsf{Ent} \quad
  \Gamma \vdash a : \mathsf{Act} \quad
  s : \mathsf{Sign} \quad
  f : \mathsf{Force}
}{
  \Gamma \vdash_{\!B} \textsf{NRM}(b,c,a,s,f)\ \mathsf{ok}
}\ \textsc{(WF-NRM)}
$$

$$
\frac{
  \Gamma \vdash e : \mathsf{Ent} \quad
  k : \mathsf{Tok} \quad
  \Gamma \vdash \varphi : \mathsf{Bool}
}{
  \Gamma \vdash_{\!B} \textsf{CLS}(e,k,\varphi)\ \mathsf{ok}
}\ \textsc{(WF-CLS)}
$$

$$
\frac{
  \Gamma \vdash h : \mathsf{Ent} \quad
  \vdash \omega : \mathcal{S}_{\text{op}} \quad
  \vdash \varepsilon : \mathrm{Eff} \quad
  n : \mathbb{N}_{\text{ev}}
}{
  \Gamma \vdash_{\!B} \textsf{PWR}(h,\omega,\varepsilon,n)\ \mathsf{ok}
}\ \textsc{(WF-PWR)}
$$

$$
\frac{
  \Gamma \vdash g : \mathsf{Bool} \quad
  \Delta \subseteq_{\text{fin}} \mathrm{Inst} \quad
  p : \mathsf{Int} \quad
  D \subseteq_{\text{fin}} \mathrm{Inst}
}{
  \Gamma \vdash_{\!B} \textsf{GRD}(g,\Delta,p,D)\ \mathsf{ok}
}\ \textsc{(WF-GRD)}
$$

$$
\frac{
  s \in \mathrm{Inst} \quad
  \vdash u : \mathsf{Iri} \quad
  m : \mathrm{Mode}
}{
  \Gamma \vdash_{\!B} \textsf{REF}(s,u,m)\ \mathsf{ok}
}\ \textsc{(WF-REF)}
\qquad
\frac{
  \vdash \phi : \mathcal{F} \quad
  u : \mathsf{Unit} \quad
  {\oplus} \in \mathrm{Cmp} \quad
  \Gamma \vdash t : \mathsf{Val}
}{
  \Gamma \vdash_{\!B} \textsf{VAL}(\phi,u,\oplus,t)\ \mathsf{ok}
}\ \textsc{(WF-VAL)}
$$

$$
\frac{
  t_{\text{text}} : \mathsf{Time} \times \mathsf{Time} \quad
  t_{\text{fact}} : \mathsf{Time} \times \mathsf{Time}
}{
  \Gamma \vdash_{\!B} \textsf{TIX}(t_{\text{text}}, t_{\text{fact}})\ \mathsf{ok}
}\ \textsc{(WF-TIX)}
$$

## 6. Metatheorems (obligations discharged in D1.5)

- **Uniqueness of Sorts.** If $\Gamma \vdash e : \tau_1$ and $\Gamma \vdash e : \tau_2$ then $\tau_1 = \tau_2$.
- **Decidability.** The relation $\Gamma \vdash e : \tau$ is decidable in time linear in $|e|$.
- **Closure (I3).** No derivation introduces a sort outside §1.
