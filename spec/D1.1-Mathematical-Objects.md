# D1.1 — Mathematical Objects

**Layer:** $\hat{K}$ (Knowledge Theory) · **Status:** DRAFT · **Depends:** D0 §3, §4

This document fixes the set-theoretic substrate of the kernel $B$. All objects are
first-order; no object denotes a function space. Notation follows D0 Appendix C.

## 1. Primitive Domains

Let the following be pairwise-disjoint, at most countable, primitive sets.

$$
\mathbb{E}\ (\text{entities}) \quad
\mathbb{I}\ (\text{IRIs}) \quad
\mathbb{K}_{\text{tok}}\ (\text{class tokens}) \quad
\mathbb{A}\ (\text{acts}) \quad
\mathbb{U}_{\text{n}}\ (\text{units})
$$

$$
\mathbb{V}\ (\text{ground values}) \quad
\mathbb{N}\ (\text{event names}) \quad
\Sigma\ (\text{lifecycle alphabet})
$$

**Axiom 1.1 (Sign).** $\text{Sign} \triangleq \{\,+,\,-\,\}$, denoting positive (duty/permission) and negative (prohibition) polarity.

**Axiom 1.2 (Force).** $\text{Force} \triangleq \{\,\mathsf{O},\,\mathsf{P},\,\mathsf{F}\,\}$ (obligation, permission, prohibition).

**Axiom 1.3 (Lifecycle).** $\Sigma \triangleq \{\,\mathsf{inforce},\,\mathsf{suspended},\,\mathsf{violated},\,\mathsf{discharged},\,\mathsf{terminated}\,\}$.

## 2. Bitemporal Index

Let $(\mathbb{T},\leq)$ be a discrete linear order (instants). A temporal *interval* is
$\iota \in \mathbb{T} \times (\mathbb{T} \cup \{\infty\})$ with $\pi_1(\iota) \leq \pi_2(\iota)$.

**Definition 2.1 (Bitemporal Index).**
$$
\text{TIX} \;\triangleq\; \{\, \langle t_{\text{text}},\, t_{\text{fact}} \rangle \mid t_{\text{text}},\, t_{\text{fact}} \in \mathbb{T}\times(\mathbb{T}\cup\{\infty\}) \,\}
$$
where $t_{\text{text}}$ is validity-in-text time and $t_{\text{fact}}$ is validity-in-fact time.

**Invariant (I6, restated).** Every instance $x \in \hat{K}$ carries a total map $\tau(x) \in \text{TIX}$.

## 3. Kernel Constructors as Relations

Each constructor is a labelled tuple space. Let $\mathrm{Expr}_T$ denote the set of
well-formed terms of the semantic language $T$ (defined in D1.3).

$$
\begin{aligned}
\text{NRM} &\subseteq \mathbb{E} \times \mathbb{E} \times \mathbb{A} \times \text{Sign} \times \text{Force} \\
\text{CLS} &\subseteq \mathbb{E} \times \mathbb{K}_{\text{tok}} \times \mathrm{Expr}_T \\
\text{PWR} &\subseteq \mathbb{E} \times \mathcal{S}_{\text{op}} \times \mathrm{Eff} \times \mathbb{N} \\
\text{GRD} &\subseteq \mathrm{Expr}_T \times \wp_{\text{fin}}(\mathrm{Inst}) \times \mathbb{Z} \times \wp_{\text{fin}}(\mathrm{Inst}) \\
\text{REF} &\subseteq \mathrm{Inst} \times \mathbb{I} \times \mathrm{Mode} \\
\text{VAL} &\subseteq \mathcal{F} \times \mathbb{U}_{\text{n}} \times \mathrm{Cmp} \times \mathbb{V}
\end{aligned}
$$

where $\mathcal{S}_{\text{op}}$ is the set of operand schemas, $\mathrm{Eff}$ the set of
effects, $\mathrm{Mode} \triangleq \{\,\mathsf{cite},\,\mathsf{amend},\,\mathsf{derogate},\,\mathsf{define}\,\}$,
$\mathrm{Cmp} \triangleq \{\,{<},\,{\leq},\,{=},\,{\geq},\,{>}\,\}$, $\mathcal{F}$ the set of
fixpoint-free value functions, and $\wp_{\text{fin}}$ the finite powerset.

**Definition 3.1 (Instances).** The set of kernel instances is the tagged union
$$
\mathrm{Inst} \;\triangleq\; \sum_{c \in B} c, \qquad B = \{\text{NRM},\text{CLS},\text{PWR},\text{GRD},\text{REF},\text{VAL},\text{TIX}\}.
$$

**Definition 3.2 (Indexed Instance).** An element of $\hat{K}$ is a pair $\langle i, \tau \rangle \in \mathrm{Inst} \times \text{TIX}$.

## 4. Registry

**Definition 4.1 (Registry).** A registry is a finite partial map
$R : \mathbb{K}_{\text{tok}} \rightharpoonup_{\text{fin}} \mathbb{V}$, versioned by $r \in \mathbb{N}$.
$R$ carries no semantics: it is a lookup table only (cf. Invariant I4).

**Axiom 4.1 (Semantic Inertness).** For any bijection $\sigma : \mathrm{dom}(R) \to \mathrm{dom}(R')$,
verdicts are invariant under $\sigma$ (proof obligation, D1.5 §T4).

## 5. Knowledge Base

**Definition 5.1 (Knowledge Base).**
$$
\hat{K} \;\triangleq\; \wp_{\text{fin}}(\mathrm{Inst} \times \text{TIX}) \times \mathrm{Reg}
$$
a finite set of indexed instances together with a registry environment $\mathrm{Reg}$.

**Axiom 5.1 (Append-Only, I2).** The only admissible mutation of $\hat{K}$ is monotone
extension $\hat{K} \sqsubseteq \hat{K}'$ produced by $\hat{E}$ (see D1.4). No in-place
deletion or overwrite exists.
